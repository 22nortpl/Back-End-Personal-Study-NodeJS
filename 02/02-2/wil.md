콜백 함수 : 다른 함수의 매개변수로 사용하는 함수
A함수 안에 B함수를 매개변수로 저장
주로 callback 키워드를 사용

다만 익명으로 callback 함수를 작성 시, callback hell에 들어갈 수도... 즉 콜백이 계속 반복되는 현상

프라미스 객체 : 말 그대로 처리 성공과 실패를 약속해둠
.then() : 성공했을 때
.catch() : 실패했을 때
프라미스 체이닝 : 이 둘을 이어서 만드는 것

async/await : async를 붙이면 그 함수 내에서는 await를 사용해서 비동기 처리가 가능
