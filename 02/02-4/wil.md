노드의 코어 모듈 : 백엔드 개발에 필요한 모듈
모듈명(기능) 설명
fs(파일 시스템) : 파일이나 폴더에 접근할 수 있는 기능 제공
http(HTTP) : HTTP 서버를 만들고 요청 처리하는 기능 제공
path(경로) : 파일 경로와 관련된 작업 기능 제공
streams(스트림) : 데이터 스트림 처리하는 기능 제공
crypto(암호화) : 암호화와 관련된 기능 제공
os(운영체제) : 운영체제와 상호작용하는 기능 제공
util(유틸리티) : 다양한 유틸리티 함수 제공
events(이벤트) : 이벤트 기반 프로그래밍 지원하는 기능 제공

cf. npm init -y가 package.json을 생성한다.

글로벌 객체 : global이라는 모듈에 포함된 객체
